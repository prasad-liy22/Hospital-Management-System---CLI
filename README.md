# Hospital Management System - CLI

A Command Line Interface (CLI) based Hospital Management System developed in C. This project was built for the Data Structures and Algorithms module to demonstrate the practical application of various data structures in a real-world scenario.

## 🚀 Features & Data Structures Implemented

This system consists of 8 main modules, each utilizing a specific data structure:

1. **Emergency Case Handling** (`Stack`) - Prioritizes the most recent critical patients (LIFO).
2. **OPD Waiting System** (`Queue`) - Manages patients in the order of their arrival (FIFO).
3. **Ward Navigation & Management** (`Doubly Linked List`) - Allows bidirectional navigation of patient records.
4. **ICU Bed Management** (`Array`) - Tracks the occupancy status of a fixed number of ICU beds.
5. **Patient Treatment Sequence** (`Singly Linked List`) - Stores the sequential treatment steps of a single patient.
6. **Round-Robin Duty Rosters** (`Circular Singly Linked List`) - Manages continuous shift assignments for medical staff.
7. **Endless Vitals Monitor** (`Circular Doubly Linked List`) - Enables seamless, endless scrolling through patient vitals in a ward.
8. **Pharmacy Inventory** (`Array` with `Bubble/Insertion Sort`) - Manages medicine stock and prioritizes dispensing based on expiration dates.
